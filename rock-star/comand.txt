1. npm install
   Описание: Устанавливает все зависимости, указанные в package.json. 
   Когда использовать: При первом запуске проекта или после добавления новых зависимостей.

2. npm run dev
   Описание: Запускает сервер разработки Next.js с горячей перезагрузкой.
   Когда использовать: Во время разработки для запуска проекта на локальном сервере.

3. npm run build
   Описание: Строит проект для продакшн, минимизирует файлы и подготавливает проект для деплоя.
   Когда использовать: Когда проект готов к деплою на продакшн сервер.

4. npm run start
   Описание: Запускает проект в продакшн-режиме, после того как он был собран с помощью команды `npm run build`.
   Когда использовать: Когда проект уже собран и готов к запуску на продакшн сервере.

5. npm install <package-name>
   Описание: Устанавливает новый пакет и добавляет его в зависимости проекта.
   Когда использовать: Для добавления новых пакетов в проект. Например, для установки lodash: `npm install lodash`.

6. npm run lint
   Описание: Запускает линтер (например, ESLint) для проверки кода на ошибки и несоответствия стандартам.
   Когда использовать: Для проверки кода на ошибки и соответствие стандартам качества.

7. npm uninstall <package-name>
   Описание: Удаляет указанный пакет из зависимостей проекта.
   Когда использовать: Когда нужно удалить пакет, который больше не используется.

8. npm list
   Описание: Показывает информацию о текущих установленных пакетах в проекте.
   Когда использовать: Для просмотра всех установленных зависимостей проекта.

9. npm update
   Описание: Обновляет все зависимости проекта до последних доступных версий.
   Когда использовать: Когда нужно обновить все пакеты до актуальных версий.

10. npm cache clean --force
    Описание: Очищает кэш npm. Полезно при возникновении проблем с установкой пакетов.
    Когда использовать: Если возникают проблемы с установкой или обновлением пакетов.
